Jujube Classification System

2021

Developed a highly accurate (98.2%) classification system for Jujube pricing using Matlab and Python, leveraging classical image processing and ML techniques tailored for agricultural product grading.

  • Feature engineering for color, texture, and size attributes.
  • Model selection with cross-validation and held-out evaluation.
  • Prototype UI for batch grading and export to CSV.
